### ComfyUI Workflow Recommendations
- **Model:** Wan 2.1 T2V (720p, 24fps, 5-second generation = 120 frames)
- **Sampling:** Use DPM++ 2M Karras, 30–40 steps, CFG 7–8
- **Negative prompt template prefix (apply to all):** `worst quality, low quality, normal quality, watermark, signature, jpeg artifacts, deformed, mutated, disfigured, blurry`
- **Batch strategy:** Generate 3 variations per scene, select best
- **Upscaling:** Optional 2× Real-ESRGAN pass for hero scenes (01, 04, 35, 36)

### Screen Capture Production
- **Prefer the live dashboard** (`make dashboard-run`) for demo scenes — it has phase indicators, real-time execution feed, safety counter, and tool chips. Much more visual than raw terminal.
- For terminal-specific scenes (launch, raw output), record with `asciinema` or OBS
- Use a custom terminal theme matching the solarpunk palette:
  - Background: `#0a1a0f` (very dark green-black)
  - Text: `#c8d6c0` (soft sage)
  - Green accent: `#4ade80`
  - Gold accent: `#fbbf24`
  - Cyan accent: `#22d3ee`
  - Red/error: `#f87171`
- Architecture diagram (Scene 07): Build in Figma or Excalidraw with matching palette
- Infographics (Scenes 26–27): Build in Figma, export as video with sequential animations

### Audio Design
- **Music:** Solarpunk ambient electronic — warm pads, gentle arpeggios, building to triumphant close. Consider artists like Tycho, Bonobo, or Ólafur Arnalds for reference tone.
- **SFX:** Subtle data-processing sounds for screen captures, gentle chimes for safety layer passes, a resonant tone for the final "agents all the way down" moment.
- **Voiceover:** Confident but warm, moderate pace (~140 WPM), slight pauses between sections.
